On this page you make a written description of how to use the structural HTML5 elements

  • Börja med att skriva upp strukturen
  • < !doctype html> - säger åt webbläsaren att det är HTML5
  • Html lang="" - här skriver du vilken språk du skriver sidan på
  • Head - här kan du bland annar skriva titel och meta data
  • body - här skriver du all din kod som skall synas på hemsidan
